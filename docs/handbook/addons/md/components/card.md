# Card (`Md::Card`)

## In plain language

A simple **Bootstrap card** block: title, body text, and a call-to-action style button — useful for dashboards or summaries.

## Source file

`addons/md/addon/components/md/card.gjs`

## Inputs

- `@title` — card heading  
- `@body` — main paragraph  
- `@action` — label on the primary button (link target is placeholder `#` in the template)  

## Logic & “services”

**Static layout only** — no services.
